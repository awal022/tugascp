# Qurban App

Aplikasi manajemen Qurban untuk penjualan dan pembelian kambing.

## Cara Jalankan

1. Clone repo:  
   `git clone https://github.com/awal022/tugascp.git`

2. Masuk folder:  
   `cd tugascp/qurban-app`

3. Setup lingkungan sesuai kebutuhan (PHP, database, dll).

4. Jalankan aplikasi.
